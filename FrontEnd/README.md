# QuizQuest Frontend

A React-based quiz application with authentication and interactive quiz features.

## Features

### Authentication
- User registration with validation
- Secure login system
- Protected routes
- JWT token management
- Automatic session handling

### Quiz Features
- Interactive quiz interface
- Timer functionality
- Progress tracking
- Score calculation
- Results dashboard

## Tech Stack

- React.js
- TypeScript
- Tailwind CSS
- React Router DOM
- Shadcn UI Components

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```
